 # 📊 Sales Analytics Dashboard

An interactive sales analytics dashboard built using Streamlit.

## 🚀 Features
- Upload CSV dataset
- Filter by Region, Category, Date
- KPI Metrics (Revenue, Profit, Quantity)
- Revenue trends over time
- Category comparison
- Region-wise profit distribution

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## 📂 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
